# Mini Laundary Order Management System 
This is a compact order management system built for dry-cleaning and laundry operations. It combines a fast FastAPI backend with a lightweight vanilla HTML/CSS/JS frontend for a simple, demo-ready workflow. Customers can create laundry orders with itemized garments, phone number, and customer name, while the system computes billing automatically from a predefined price catalog.  The project supports full order lifecycle management with status transitions from RECEIVED to PROCESSING, READY, and DELIVERED. It includes a dashboard that displays total orders, revenue, status breakdown, and garment revenue metrics. Orders can be filtered by customer name and status, and individual order details open in a modal for convenient review and updates.
